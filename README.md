# Simple Port Scanner

## Overview

This is a basic cybersecurity tool written in Python that scans common ports on a target server or website.

The program checks whether specific ports are open or closed.

## Features

* Scan a target IP or domain
* Detect open and closed ports
* Uses Python socket library

## Example

Input:
google.com

Output:
Port 80 is OPEN
Port 443 is OPEN

## Technologies Used

* Python
* Socket Programming
